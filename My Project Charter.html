🛠️ 9. Project Methodology

My methodology is a predictive/traditional approach. This approach would work best for my project because I have all of the requirements gathered and documented before development, my requirements are set in stone, and it will be a single delivery at the end of the project.

🛠️ 10. Project Tool Selection

I chose Excel/Google Sheets because my methodology is predictive. Tools like Jira, Trello, and Notion wouldn’t fit with my methodology because they are more suited for adaptive/hybrid approaches. MS Project and Monday.com are also great to use for predictive methodology, but the reason I chose Excel is because I have more experience with MS Excel.
